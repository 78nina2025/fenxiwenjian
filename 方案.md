## 四、规则 Schema(与混沌/江恩项目 24 字段对齐)

```
rule_id, name, category_l1, category_l2, sub_category,
direction, timeframe, applicability, preconditions,
trigger, trigger_dsl, confirmation, invalidation,
dow_phase, trend_level, confirm_index,
target_logic, stop_logic,
source_book, source_quote,
authors_confidence, rule_weight, remark
```

道氏专用字段:
- `dow_phase`: accumulation / public_participation / distribution / panic;
- `trend_level`: primary_bull / primary_bear / secondary / minor / line;
- `confirm_index`: 需要同步验证的另一指数(DJTA / 深证成指 / 北向净流入等)。

## 五、流水线

```
01_extract_text.py   PDF/EPUB/AZW3/MOBI/DOC → JSONL(逐页文本)
02_clean.py          去页眉页脚 / 合并段落 / 去重
03_extract_rules.py  Claude Sonnet 4.6 按 schema 抽取
04_merge.py          (category_l2, sub_category, direction) 分桶 + 相似度去重
05_export.py         merged JSONL → xlsx + docx
```

电子书格式处理:
- EPUB: 用 `ebooklib` 解析 XHTML → 纯文本;
- AZW3/MOBI/AZW: 先 `calibre ebook-convert` → EPUB,再走上面流程;
- DOC: `antiword` 或 `python-docx2txt` 提取。
