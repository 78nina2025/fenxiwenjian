## 3  混沌/分形体系知识地图

混沌与分形在金融中的应用可以分为 **理论派** 和 **交易派** 两条主线:

### 3.1  理论派(How to See 市场)

```
非线性动力学
  ├── 混沌理论:对初始条件敏感(蝴蝶效应)、奇异吸引子、Lyapunov 指数
  ├── 分形几何:自相似、分形维数、Hurst 指数
  ├── 复杂自适应系统:涌现、协同、相变
  └── 金融应用
        ├── FMH (Fractal Market Hypothesis) — Peters
        ├── AMH (Adaptive Market Hypothesis) — Lo
        ├── Fractional Brownian Motion — Mandelbrot
        └── Multifractal 模型 — Kantelhardt, Calvet
```

### 3.2  交易派(How to Act 操盘)

```
Bill Williams "Profitunity" 五维度
  ├── 1. 空间维(Phase Space)  — Alligator 三线(13/8/5 SMMA,错位 8/5/3)
  ├── 2. 能量维(Energy)        — Awesome Oscillator (AO)
  ├── 3. 力量维(Force)         — Accelerator/Decelerator (AC)
  ├── 4. 区域维(Zone)          — AO + AC 同色,区分买卖区
  └── 5. 平衡维(Balance Line) — 分形(Fractals)+ 平衡线突破
Five Wise Men (5 位智者) — 五种入场信号
短线工具:Divergent Bar、Angulation、Squat Bar
心态管理:《Trading Chaos II》第 1-3 章
```

### 3.3  A 股本土化改造

- 噪声趋势策略(IF/IC/IH 股指期货,论文来源)
- T+1 与涨跌停下分形信号的延迟确认
- Alligator 参数在日线/周线上的 A 股最优化

## 4  规则 Schema(混沌专版)

与江恩、量价 schema 同构,但加入 **fractal_dim / hurst / lyapunov / regime_chaos** 等字段。

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `rule_id` | string | Y | C-00001..C-NNNNN |
| `name` | string | Y | 中文简称 ≤25 字 |
| `category_l1` | enum | Y | thought / principle / rule |
| `category_l2` | enum | Y | profitunity / fmh / chaos_math / risk / psychology / arb_strategy |
| `sub_category` | string | Y | alligator / fractal / ao / ac / hurst / lyapunov / zone / divergent |
| `direction` | enum | Y | long / short / neutral / exit / both |
| `timeframe` | enum | Y | intraday / daily / weekly / monthly / multi |
| `applicability` | string | Y | 股票 / 期货 / 外汇 / 通用 |
| `preconditions` | string | Y | 前置条件(如 market_regime = trending) |
| `trigger` | string | Y | 触发条件 |
| `trigger_dsl` | string | N | 可执行表达式 |
| `confirmation` | string | N | 确认条件(如第 5 根 bar 收盘) |
| `invalidation` | string | Y | 失效条件 |
| `hurst_band` | string | N | Hurst 指数适用带(如 H∈[0.55,0.70]) |
| `fractal_dim` | string | N | 分形维数条件 |
| `lyapunov_sign` | string | N | 正/负/零 |
| `chaos_regime` | enum | N | persistent / antipersistent / random / chaotic |
| `target_logic` | string | N | 目标位算法 |
| `stop_logic` | string | N | 止损算法 |
| `win_rate_expected` | float | N | 原著声称胜率 |
| `source_book` | string | Y | 来源书名 |
| `source_chapter` | string | N | 章节 |
| `source_quote` | string | Y | ≤200 字摘录 |
| `authors_confidence` | float | Y | 原著置信度(0-1) |
| `rule_weight` | float | Y | 系统打分权重 |
| `remark` | string | N | 备注 |
